## 🔧 技能说明

### 技能1：tender-browser-scraper（实际爬取工具）

**路径**：`/skills/tender-browser-scraper/`

**类型**：Node.js脚本

**核心工具**：Playwright + Scrapling

**功能**：
- ✅ 真实浏览器模拟
- ✅ 智能反爬绕过（Cloudflare、创宇盾）
- ✅ 验证码人工介入（遇到验证码暂停并通知）
- ✅ 双重模式切换（Playwright失败→Scrapling）

**支持网站**：
- ✅ 重庆市公共资源交易中心（https://www.cqggzy.com）

**可扩展**（配置即可）：

- 贵州省公共资源交易中心
- 云南省公共资源交易中心

## 💻 使用方法

### 基础用法

```bash
# 抓取重庆公共资源交易中心
node ~/.openclaw/workspace/skills/tender-browser-scraper/scripts/scrape.js --site cqggzy

# 指定输出目录
node ~/.openclaw/workspace/skills/tender-browser-scraper/scripts/scrape.js --site cqggzy --output ./output/tenders

# 设置最大项目数
node ~/.openclaw/workspace/skills/tender-browser-scraper/scripts/scrape.js --site cqggzy --max 50
```

### 执行流程

1. Playwright启动浏览器
2. 访问招标网站
3. 模拟真实用户行为（滚动、点击）
4. 抓取招标列表
5. 提取项目详情
6. 智能过滤（关键词匹配）
7. 保存到JSON文件

## ⚠️ 可能遇到的问题

### 问题1：Playwright未安装

**现象**：
```
Error: Playwright not found
```

**解决方案**：
```bash
npm install -g playwright
playwright install chromium
```

---

### 问题2：反爬机制拦截

**现象**：
```
Error: Cloudflare detected
```

**解决方案**：
- ✅ 技能会自动切换Scrapling模式
