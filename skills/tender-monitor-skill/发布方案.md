## 📋 发布步骤

### 第一步：完善代码
```bash
# 创建skill目录
cd ~/.openclaw/skills/
mkdir tender-monitor
cd tender-monitor

# 初始化package.json
npm init -y

# 安装依赖
npm install playwright scrapling axios
```

### 第二步：编写核心代码
创建 `src/index.js`：
```javascript
class TenderMonitor {
  constructor(config) {
    this.config = config;
    this.crawler = new Crawler(config.sites);
    this.filter = new Filter(config.keywords, config.company);
    this.analyzer = new Analyzer(config.company);
    this.reporter = new Reporter(config.output);
    this.notifier = new Notifier(config.notification);
  }

  async run() {
    // 1. 爬取数据
    const projects = await this.crawler.fetch();
    
    // 2. 筛选项目
    const filtered = await this.filter.apply(projects);
    
    // 3. 分析项目
    const analyzed = await this.analyzer.analyze(filtered);
    
    // 4. 生成报告
    const report = await this.reporter.generate(analyzed);
    
    // 5. 发送通知
    await this.notifier.send(report);
    
    return report;
  }
}

module.exports = TenderMonitor;
```

### 第三步：编写测试

创建 `tests/crawler.test.js`：
