# 错误教训技能 - Error Lessons

**用途**：记录和预防重复错误，形成可执行的工作规则

## 如何使用这个技能

1. **犯错时**：在 `lessons/` 目录下创建新的教训文档
2. **定期回顾**：每周查看教训文档，更新规则
3. **新任务前**：检查相关教训，避免重复错误

## 教训文档模板

创建新教训时，使用以下模板：

```markdown
# [日期] [简短标题]

**错误类型**：配置/逻辑/执行/沟通  
**严重程度**：🔴 高 / ⚠️ 中 / 🟡 低

## 问题现象
[描述用户看到的问题]

## 根本原因
[分析为什么会出现这个问题]

## 影响范围
[哪些功能/用户受影响]

## 解决方案
[具体修复步骤]

## 预防规则
[未来如何避免]

**相关文件**：
- [列出相关配置文件]

**教训标签**：#配置 #定时任务 #验证
```
