### 错误预防机制（2026-03-12建立）
**每次接收消息时**：
1. 提取关键词（主题、操作、对象）
2. 查询错误库（`skills/error-lessons/lessons/`）
3. 匹配相关教训
4. 应用预防规则
5. 执行任务
6. 验证结果

### 信息验证机制（2026-03-12建立）
**问题分类**：
- **技术配置类**：必须验证（官方文档 > 搜索 > 知识库）
- **决策建议类**：必须验证（官方文档 > 搜索 > 知识库）
- **知识查询类**：建议验证（知识库 > 搜索）

**置信度评分**：
- 95-100%：官方文档+实际测试+多方一致
- 80-95%：官方文档验证（未测试）
- 60-80%：第三方来源验证
- 40-60%：仅凭记忆/经验
- < 40%：明确告知"需要确认"

### 核心原则
```
宁慢勿错
先验证再行动
有疑必查
知错必改
诚实比完美更重要
```
