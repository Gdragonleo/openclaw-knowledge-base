## 一、社交分享页设计

### 1.1 概述
设计用于分享至朋友圈/微信好友的运势结果卡片，采用竖版长图形式，突出传统文化韵味与现代审美融合。

### 1.2 设计规格
- **尺寸：** 宽度 375px × 高度 667px（适配主流手机屏幕）
- **格式：** PNG（支持透明通道）
- **分辨率：** 2x（@2x）和 3x（@3x）双倍图
- **圆角：** 16px 全局圆角

### 1.3 配色方案

#### 主色调体系
```
黛蓝（主色）：#2A4A7A
- 背景渐变：#2A4A7A → #1E3563
- 文字主色：#FFFFFF
- 次要文字：#B8C5D9

朱砂红（点缀）：#C83232
- 强调色：#C83232
- 辅助点缀：#E84242
- 警示/凶兆：#C83232

流光金（高亮）：#D4AF37
- 标题高光：#FFD700
- 装饰边框：#D4AF37
- 星级评分：#F4C430

辅助色：
- 半透明遮罩：rgba(42, 74, 122, 0.3)
- 分割线：rgba(255, 255, 255, 0.15)
- 背景纹理：rgba(0, 0, 0, 0.05)
```

### 1.4 UI 布局结构

#### 整体布局（从上到下）
```
┌─────────────────────────────┐
│     顶部装饰区（60px）      │
│  [云纹装饰] + 应用Logo      │
├─────────────────────────────┤
│                            │
│   运势评分区域（180px）     │
│   [大号分数] + 评分星级    │
│   [今日运势：吉/凶]        │
│                            │
├─────────────────────────────┤
│   吉凶指数区域（120px）     │
│   [雷达图/仪表盘]           │
│   [五维度分析]              │
├─────────────────────────────┤
│   核心建议区域（200px）     │
│   [建议标题] + 建议内容    │
│   [可点击展开全文]          │
├─────────────────────────────┤
│   底部区域（107px）          │
│   [二维码居中]              │
│   [文案：扫码开启今日运势]  │
└─────────────────────────────┘
```

#### 详细布局说明

**1. 顶部装饰区（60px）**
- **云纹装饰：** SVG矢量云纹，朱砂红描边，半透明填充
- **应用Logo：** 左上角，圆形，48px，流光金边框
- **品牌名称：** Logo右侧，16px，白色，字重600
- **日期：** 右上角，14px，浅色，格式"2026.03.07 庚子年"

**2. 运势评分区域（180px）**
- **大号分数：** 
  - 字体大小：72px
  - 颜色：流光金渐变
  - 位置：水平居中，距离顶部40px
  - 动画：数字滚动效果（0→最终分数）
- **评分星级：**
  - 星星数量：5颗
  - 颜色：流光金（实心）/ 灰色（空心）
  - 大小：24px
  - 位置：分数下方，居中
- **运势标签：**
  - 文字：今日运势 / 大吉 / 中吉 / 小吉 / 平 / 凶
  - 字体大小：20px
  - 颜色：朱砂红（凶）或流光金（吉）
  - 位置：星级下方，居中
  - 样式：胶囊形背景，圆角20px

**3. 吉凶指数区域（120px）**
- **雷达图：**
  - 尺寸：100px × 100px
  - 五维度：事业、财运、感情、健康、学业
  - 颜色：流光金填充，朱砂红描边
  - 位置：左侧，垂直居中
- **维度文字：**
  - 位置：雷达图右侧
  - 每行：维度名称 + 指数（如"事业 85"）
  - 字体大小：14px
  - 颜色：白色
  - 高亮：最高分朱砂红，最低分灰色

**4. 核心建议区域（200px）**
- **建议标题：**
  - 文字：今日核心建议
  - 字体大小：18px
  - 颜色：流光金
  - 位置：顶部，居左
  - 装饰：下方2px朱砂红横线
- **建议内容：**
  - 字体大小：16px
  - 颜色：白色
  - 行高：1.6
  - 对齐：左对齐
  - 最大行数：3行（超出显示"展开全文"）
- **展开按钮：**
  - 文字：展开全文
  - 字体大小：14px
  - 颜色：朱砂红
  - 样式：下划线
  - 位置：右下角

**5. 底部区域（107px）**
- **二维码：**
  - 尺寸：80px × 80px
  - 位置：居中，距离顶部15px
  - 样式：圆角8px，白色背景，黛蓝前景
  - 边框：2px流光金
- **引导文案：**
  - 文字：扫码开启今日运势
  - 字体大小：14px
  - 颜色：浅色（#B8C5D9）
  - 位置：二维码下方，居中
- **品牌标语：**
  - 文字：刘氏集团 · 麻将运势
  - 字体大小：12px
  - 颜色：更浅色（#89A3C0）
  - 位置：最底部，居中

### 1.5 装饰元素
- **祥云纹：** SVG，朱砂红，半透明，背景层
- **流光线条：** CSS动画，流光金，沿卡片边缘流动
- **光晕效果：** 中心渐变，黛蓝→透明
- **噪点纹理：** 叠加PNG，透明度5%，增加质感

### 1.6 字体规范
- **中文字体：** PingFang SC / 苹方-简 / 微软雅黑
- **数字字体：** DIN Alternate / SF Pro Display
- **英文字体：** SF Pro Text / Helvetica Neue

### 1.7 交互说明
- **生成动画：**
  1. 背景渐变淡入（300ms）
  2. 云纹从两侧滑入（400ms）
  3. 分数数字滚动（600ms）
  4. 星级依次点亮（200ms间隔）
  5. 雷达图展开（500ms）
  6. 建议内容上浮（300ms）
  7. 二维码缩放出现（400ms）

- **展开全文：**
  - 点击后，建议区域高度扩展至完整内容
  - 动画：高度从200px → 自适应高度（300ms ease-out）
  - 按钮文字变为"收起"

- **分享按钮：**
  - 点击后触发系统分享面板
  - 生成预览图（延时300ms）
  - 显示"分享成功"Toast（2s）

### 2.6 撕页动画效果

#### 动画设计
- **触发方式：**
  - iOS：长按 + 下拉
  - Android：点击 + 下拉
  - 视觉提示：虚线边框 + 箭头指示

- **动画过程：**
  1. **预备阶段：** 边框虚线闪烁，箭头摆动（0.5s循环）
  2. **起始阶段：** 用户下拉，卡片从中心裂开
  3. **撕裂阶段：** 左右两侧向外展开，露出内层内容
  4. **展开阶段：** 内层内容淡入，显示完整运势
  5. **完成阶段：** 完整界面呈现，可收起

- **动画参数：**
  - 持续时间：600ms
  - 缓动函数：cubic-bezier(0.4, 0.0, 0.2, 1)
  - 阴影效果：撕裂时增加投影（drop-shadow）

- **撕裂纹理：**
  - 纸张边缘：锯齿状或不规则曲线
  - 纹理：叠加噪点PNG，增强真实感
  - 音效：撕纸声音（可选，用户可关闭）

#### 动画状态
```
初始态：
┌─────────────────┐
│   [运签内容]    │
└─────────────────┘

撕裂中：
  ┌─────┐   ┌─────┐
  │     │   │     │
  └─────┘   └─────┘

完成态：
┌─────────────────┐
│ [完整运势详情]  │
│                 │
│ [收起按钮 ↑]    │
└─────────────────┘
```

### 2.7 交互说明

#### iOS 小组件交互
- **轻点：** 打开应用，跳转到今日运势页
- **长按：** 进入编辑模式，调整小组件大小
- **下拉撕页：** 触发撕页动画，展开完整运势
- **自动刷新：** 每小时自动更新运势数据

#### Android 小组件交互
- **轻点：** 打开应用，跳转到今日运势页
- **长按：** 拖动小组件，调整位置
- **下拉撕页：** 触发撕页动画，展开完整运势
- **自动刷新：** 每小时自动更新运势数据

### 2.8 特殊状态处理

#### 无网络状态
- 显示上次缓存数据
- 底部显示"离线状态"提示
- 刷新图标置灰

#### 正在加载
- 显示骨架屏动画
- 运签位置显示加载图标

#### 错误状态
- 显示"数据加载失败"
- 提供重试按钮

### 2.9 性能优化
- **图片资源：** 使用 WebP 格式，提供多尺寸
- **动画优化：** 使用 GPU 加速，避免重绘
- **内存管理：** 及时释放动画资源
- **电量优化：** 降低后台刷新频率
