# 麻将运势微信小程序 UI设计规范

**版本：** 1.0  
**更新日期：** 2026年3月7日  
**设计者：** 刘氏集团UI设计师  
**适用项目：** 麻将运势微信小程序（独立产品）

---

## 1. 设计定位与风格

### 1.1 设计关键词
- **国潮麻将风**：融合传统麻将元素与现代设计语言
- **传统文化**：体现中国传统文化底蕴
- **趣味性**：轻松愉快的用户体验
- **神秘感**：运势预测的神秘氛围

### 1.2 目标用户画像
- **年龄层**：25-40岁
- **特征**：爱打麻将的娱乐型用户，对传统文化有一定兴趣
- **使用场景**：麻将局前查看运势、社交分享、日常娱乐

## 3. 字体规范

### 3.1 字体选择
- **标题字体**：衬线字体（推荐：方正宋体、思源宋体）
- **正文字体**：系统默认字体（微信小程序默认字体）
- **特殊字体**：可考虑引入一款书法字体用于装饰性标题

### 3.2 字号体系（rpx单位）
| 层级 | 字号(rpx) | 行高(rpx) | 使用场景 |
|------|-----------|-----------|----------|
| **H1** | 48 | 68 | 页面主标题 |
| **H2** | 36 | 50 | 模块标题 |
| **H3** | 32 | 44 | 卡片标题 |
| **Body** | 28 | 40 | 正文内容 |
| **Caption** | 24 | 34 | 辅助说明文字 |
| **Small** | 20 | 28 | 标签文字 |

### 3.3 字重规范
- **Regular** (400)：正文内容
- **Medium** (500)：次要标题
- **Bold** (700)：主要标题、强调文字

## 6. 交互动效说明

### 6.1 进场动画
1. **罗盘旋转进场**
   - 持续时间：800ms
   - 效果：从0°旋转到360°，配合缓动函数
   - 触发时机：页面加载完成时

2. **评分数字跳动**
   - 持续时间：1200ms
   - 效果：数字从0滚动到目标分数
   - 触发时机：罗盘动画完成后

### 6.2 微交互
1. **点击方位弹出详情**
   - 触发区域：罗盘的四个方位扇形区
   - 反馈效果：轻微缩放(scale: 1.05) + 颜色加深
   - 弹出内容：该方位的详细运势解读

2. **卡片悬停效果**
   - 触发条件：手指悬停（移动端为点击态）
   - 效果：轻微上浮 + 阴影加深
   - 持续时间：200ms

### 6.3 页面过渡
- **页面切换**：右滑进入，左滑退出
- **返回动画**：渐隐 + 左移
- **加载状态**：骨架屏 + 旋转加载图标

## 7. 微信小程序适配规范

### 7.1 安全区域处理
```
iPhoneX及全面屏适配：
┌─────────────────────────────────────┐
│ 状态栏 (44px)                       │
├─────────────────────────────────────┤
│ 内容区域                            │
│                                     │
│                                     │
│                                     │
├─────────────────────────────────────┤
│ 底部安全区域 (34px)                 │
│ + TabBar高度 (98px)                 │
└─────────────────────────────────────┘

实现方案：
1. 使用wx.getSystemInfoSync()获取安全区域
2. CSS变量：--safe-area-inset-bottom
3. 页面内容padding-bottom: calc(env(safe-area-inset-bottom) + 98rpx)
```

### 7.2 底部TabBar设计（如有）
```
尺寸规范：
- 高度：98rpx
- 图标：54rpx × 54rpx
- 文字：20rpx
- 选中状态：麻将绿 #2E8B57
- 未选中状态：中灰 #666666

建议Tab项：
1. 首页（运势）
2. 历史记录
3. 我的
```

### 7.3 胶囊按钮区域规避
```
顶部右侧规避区域：
┌─────────────────────────────────────┐
│                 ┌─────┐             │
│                 │胶囊 │             │
│                 │按钮 │             │
│                 └─────┘             │
├─────────────────────────────────────┤
│                                     │
│       内容区域（避开44×44区域）     │
│                                     │
└─────────────────────────────────────┘

实现方案：
1. 获取胶囊按钮位置：wx.getMenuButtonBoundingClientRect()
2. 内容区域padding-right: 胶囊按钮宽度 + 10rpx
3. 重要操作按钮避开该区域
```

### 7.4 响应式适配
- **基础单位**：使用rpx（responsive pixel）
- **设计稿基准**：750rpx = 375px（iPhone6/7/8）
- **字体适配**：使用rpx单位，系统自动缩放
- **图片适配**：使用mode="aspectFit"或"aspectFill"

## 8. 图标与资源规范

### 8.1 图标尺寸
| 使用场景 | 尺寸(rpx) | 格式 |
|---------|-----------|------|
| TabBar图标 | 54×54 | PNG/SVG |
| 功能图标 | 48×48 | PNG/SVG |
| 装饰图标 | 64×64 | PNG/SVG |
| 小程序码 | 200×200 | PNG |

### 8.2 图片资源
