# 麻将算命微信小程序 - 牌型 & 花色命盘展示 UI 设计方案

## 项目概述
- **项目名称**: 麻将算命微信小程序
- **游戏类型**: 四川血战到底
- **任务**: 设计"牌型 & 花色命盘"展示 UI
- **设计目标**: 将算命结果（近期适合的牌型和花色）以视觉化方式展示，融合国风神秘感与微信小程序简洁风格
- **设计完成时间**: 2026-03-07

---

## 1. 结果展示页 UI 布局

### 1.1 页面整体结构
```
├── 头部区域 (Header)
│   ├── 返回按钮
│   ├── 页面标题："您的牌型命盘"
│   └── 分享按钮
├── 主内容区 (Main Content)
│   ├── 命盘圆形图 (中心视觉)
│   ├── 花色扇形分布
│   ├── 牌型信息卡片
│   └── 运势描述区
└── 底部操作区 (Footer)
    ├── 保存结果按钮
    └── 再次测算按钮
```

### 1.2 命盘圆形图组件设计
**组件名称**: MahjongFortuneWheel
**层级结构**:
```
FortuneWheelContainer
├── WheelBase (圆形底板)
├── ColorSegments (三个120°扇形)
│   ├── CharacterSegment (万 - 红色系)
│   ├── DotSegment (筒 - 绿色系)
│   └── BambooSegment (条 - 蓝色系)
├── MahjongTileIcons (花色图标)
│   ├── CharacterIcon (万图标)
│   ├── DotIcon (筒图标)
│   └── BambooIcon (条图标)
├── FortuneIndicator (运势指针)
└── WheelCenter (中心装饰)
```

### 1.3 牌型信息卡片组件
**组件名称**: HandTypeCard
**层级结构**:
```
HandTypeCardContainer
├── CardHeader (卡片头部)
│   ├── HandTypeIcon (牌型图标)
│   └── HandTypeName (牌型名称)
├── CardBody (卡片主体)
│   ├── HandTypeDescription (牌型描述)
│   ├── WinningProbability (胡牌概率)
│   └── DifficultyLevel (难度等级)
└── CardFooter (卡片底部)
    └── Tips (小贴士)
```

### 1.4 运势描述区组件
**组件名称**: FortuneDescription
**层级结构**:
```
FortuneDescriptionContainer
├── TimeRange (运势时间范围)
├── OverallFortune (总体运势评价)
├── DetailedAnalysis (详细分析)
│   ├── CharacterFortune (万花色运势)
│   ├── DotFortune (筒花色运势)
│   └── BambooFortune (条花色运势)
└── Recommendations (建议)
```

### 1.5 交互说明
1. **命盘圆形图交互**:
   - 点击扇形区域: 高亮该花色，显示详细运势
   - 长按命盘: 显示运势解读弹窗
   - 滑动旋转: 可360°旋转查看不同角度

2. **牌型卡片交互**:
   - 点击卡片: 展开/收起详细描述
   - 滑动切换: 左右滑动查看其他推荐牌型

3. **运势描述区交互**:
   - 下拉刷新: 重新获取运势分析
   - 点击建议项: 跳转到对应攻略页面

## 2. 配色方案

### 2.1 主色调体系
**设计理念**: 以麻将传统色彩为基础，结合国风水墨意境

#### 2.1.1 花色对应配色
1. **万 (Characters) - 红色系**:
   - 主色: #C62828 (朱砂红)
   - 辅色: #EF5350 (珊瑚红)
   - 浅色: #FFCDD2 (樱花粉)
   - 深色: #B71C1C (绛红)

2. **筒 (Dots) - 绿色系**:
   - 主色: #2E7D32 (竹青绿)
   - 辅色: #66BB6A (翡翠绿)
   - 浅色: #C8E6C9 (嫩竹绿)
   - 深色: #1B5E20 (墨绿)

3. **条 (Bamboo) - 蓝色系**:
   - 主色: #1565C0 (青花蓝)
   - 辅色: #42A5F5 (天青蓝)
   - 浅色: #BBDEFB (淡天蓝)
   - 深色: #0D47A1 (藏青蓝)

#### 2.1.2 中性色调
- **背景色**: #FAF9F6 (宣纸白)
- **文字主色**: #2C3E50 (墨黑)
- **文字辅色**: #7F8C8D (灰墨)
- **边框色**: #E0E0E0 (淡灰)
- **强调色**: #F57C00 (琥珀橙)

#### 2.1.3 功能色
- **成功色**: #4CAF50 (翠绿)
- **警告色**: #FF9800 (橙黄)
- **错误色**: #F44336 (朱红)
- **信息色**: #2196F3 (天蓝)

### 2.2 配色应用指南
1. **命盘圆形图**:
   - 扇形区域使用各花色主色
   - 渐变效果: 主色 → 辅色径向渐变
   - 边框: 深色描边，1px宽度

2. **牌型卡片**:
   - 背景: 宣纸白 #FAF9F6
   - 边框: 淡灰 #E0E0E0
   - 标题: 强调色 #F57C00
   - 内容: 墨黑 #2C3E50

3. **运势描述区**:
   - 背景: 白色 #FFFFFF
   - 分割线: #EEEEEE
   - 重点文字: 各花色对应主色

## 3. 视觉风格定义

### 3.1 国风神秘感元素
1. **纹理应用**:
   - 背景: 宣纸纹理叠加
   - 装饰: 云纹、水纹、回纹边框
   - 图标: 采用传统纹样简化设计

2. **图形风格**:
   - 圆角设计: 采用大圆角 (16-24px)
   - 阴影效果: 柔和阴影，模拟宣纸层次
   - 渐变过渡: 水墨渐变效果

3. **字体选择**:
   - 标题字体: 思源宋体 / 方正楷体
   - 正文字体: 苹方 / 思源黑体
   - 数字字体: DIN Alternate

### 3.2 微信小程序简洁风格
1. **布局原则**:
   - 留白充分: 使用微信小程序标准间距
   - 对齐严谨: 网格系统对齐
   - 层级清晰: 通过大小、颜色、间距区分层级

2. **组件规范**:
   - 按钮: 使用微信标准按钮样式
   - 卡片: 圆角 + 阴影 + 内边距
   - 列表: 简洁分割线设计

3. **动效设计**:
   - 转场动画: 淡入淡出，时长300ms
   - 交互反馈: 轻微缩放/颜色变化
   - 加载动画: 麻将旋转动画

### 3.3 风格融合方案
1. **色彩融合**:
   - 以国风色彩为主，微信标准色为辅
   - 饱和度控制: 国风色彩稍降饱和度

2. **元素融合**:
   - 国风纹理作为背景层
   - 微信标准组件作为交互层
   - 传统纹样作为装饰元素

3. **情感传达**:
   - 神秘感: 通过渐变、纹理、朦胧效果
   - 亲切感: 通过圆角、柔和色彩、简洁布局
   - 专业感: 通过严谨对齐、清晰层级

## 4. 详细组件规范

### 4.1 MahjongFortuneWheel 详细参数
- **尺寸**: 280px × 280px (小程序中)
- **扇形角度**: 120° (三等分)
- **图标尺寸**: 40px × 40px
- **指针长度**: 120px
- **旋转动画**: ease-in-out, 0.5s

### 4.2 HandTypeCard 详细参数
- **尺寸**: 100%宽度，高度自适应
- **内边距**: 24px
- **圆角**: 16px
- **阴影**: 0 4px 12px rgba(0,0,0,0.08)
- **展开高度**: 200px

### 4.3 FortuneDescription 详细参数
- **行高**: 1.6
- **段落间距**: 16px
- **重点文字**: 加粗 + 花色对应颜色
- **分割线**: 1px solid #EEEEEE
