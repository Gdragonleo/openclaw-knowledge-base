## 2026-03-13 21:49 - network_error 根本原因分析

**错误**：每30分钟查询协作任务连续失败12次
**场景**：定时任务自动执行
**原因**：
1. **调用频率过高**：每30分钟一次（每天48次）
2. **飞书API限流**：触发400错误，3. **隔离会话模式**：网络不稳定时容易失败且无法恢复
4. **没有错误恢复机制**：连续失败12次

**影响**：
- 🔴 导致你今天一直收到"飞书喊你不答应"的反馈
- 🔴 其他正常任务被掩盖

**正确做法**：
1. 降低调用频率（每30分钟 → 每1小时或每2小时）
2. 改为主会话模式（isolated → main）
3. 添加错误恢复机制（连续错误3次后暂停并发送告警）

**状态**：✅ 已停止所有定时任务，**人物**：小八爪
