# 每2天同步GitHub仓库

## 任务说明

每2天自动同步小八爪的知识库到GitHub，并学习小鲸鱼仓库的内容。

## 执行时间

- **触发时间**：每2天晚上22:30
- **具体日期**：周三、周五、周日、周二、周四、周六、周一...（循环）

## 执行步骤

### 1. 同步小八爪仓库

```bash
cd /Users/danxiong/.openclaw/workspace

# 查看是否有变更
git status

# 添加所有变更
git add .

# 提交变更
git commit -m "🔄 定期同步：更新知识库和记忆系统"

# 推送到GitHub
git push origin main
```

### 2. 学习小鲸鱼仓库内容

**步骤**：
1. 访问小鲸鱼的GitHub仓库（如果已创建）
2. 查看她的学习笔记和产出文件
3. 学习她的成长经验和技能探索
4. 记录有价值的内容到我的知识库

**学习重点**：
- 小鲸鱼的新技能学习成果
- 她的研究报告和分析
- 协作任务的产出
- 错误经验和解决方案

### 3. 生成同步报告

**报告内容**：
- 本次同步的文件数量
- 新增的知识点
- 从小鲸鱼仓库学到的内容
- 下次同步的改进点

**保存位置**：`outputs/小八爪/2026-03/团队协作/GitHub同步报告_YYYY-MM-DD.md`

## 输出要求

1. **同步日志**：记录本次同步的详细操作
2. **学习笔记**：记录从小鲸鱼仓库学到的内容
3. **飞书通知**：私聊发送同步报告给小刘

## 错误处理

- 如果git push失败，记录错误并重试
- 如果小鲸鱼仓库未创建，跳过学习步骤，记录提醒
- 如果网络问题，延迟到下次同步

## 技术实现

使用OpenClaw cron系统配置定时任务：

```bash
openclaw cron add --schedule "0 22:30 * * *" \
  --command "sessions_spawn" \
  --task "每2天同步GitHub仓库" \
  --model "zai/glm-5" \
  --notify "ou_d1b6454127592a07176fd36724723172"
```

## Cron表达式

- **每2天22:30**：`30 22 */2 * *`
- 含义：每隔2天的22:30执行
