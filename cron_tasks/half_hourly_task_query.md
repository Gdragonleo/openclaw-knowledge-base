## 错误处理

### 如果查询失败
1. 记录错误到 memory/错误日志.md
2. 私聊通知小刘
3. 30分钟后自动重试

### 如果任务执行失败
1. 更新任务状态为"🔄 进行中"
2. 记录错误详情
3. 尝试重新执行（最多3次）
4. 3次失败后标记为"❌ 已取消"并通知小刘
