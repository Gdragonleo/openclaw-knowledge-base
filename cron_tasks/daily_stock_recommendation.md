# 每日A股推荐任务

## 任务配置

**任务ID**: daily_a_stock_recommendation  
**触发时间**: 每个交易日 9:15  
**风险偏好**: 激进型  
**资金规模**: 5-20万

## 执行步骤

### 第一步：数据采集（9:00-9:10）

**数据源**：
1. **AkShare**（Python库，免费）
   - A股实时行情
   - 财务数据
   - 资金流向

2. **新浪财经**（爬虫备份）
   - 板块热点
   - 龙虎榜数据
   - 北向资金

**数据清单**：
- 全市场A股行情（约5000只）
- 财务指标（PE/PB/ROE/营收增长）
- 技术指标（MA/MACD/RSI/成交量）
- 资金数据（北向/龙虎榜/主力资金）
- 热点板块（涨幅/资金流入）

---

### 第二步：策略筛选（9:10-9:13）

#### 策略1：价值投资（2只）

**筛选条件**：
```python
# 激进型价值投资
value_stocks = stocks[
    (stocks['pe'] < 20) &  # 适度低估（激进可放宽）
    (stocks['roe'] > 15) &  # 高ROE
    (stocks['dividend_yield'] > 2) &  # 分红不错
    (stocks['profit_growth'] > 20)  # 利润增长
].sort_values('pe').head(5)
```

**推荐数量**：2只（高分红 + 低估值）

---

#### 策略2：成长股（3只）

**筛选条件**：
```python
# 高成长潜力
growth_stocks = stocks[
    (stocks['revenue_growth'] > 40) &  # 营收高增长
    (stocks['profit_growth'] > 50) &  # 利润高增长
    (stocks['industry'] in ['新能源', '半导体', '医药', 'AI']) &  # 热门行业
    (stocks['market_cap'] < 800亿)  # 中小市值，成长空间大
].sort_values('profit_growth').head(5)
```

**推荐数量**：3只（高增长 + 热门赛道）

---

#### 策略3：技术分析（2只）

**筛选条件**：
```python
# 技术面突破
tech_stocks = stocks[
    (stocks['price'] > stocks['ma20']) &  # 站上20日线
    (stocks['macd_golden_cross'] == True) &  # MACD金叉
    (stocks['volume_ratio'] > 1.5) &  # 放量
    (stocks['rsi'] > 50) & (stocks['rsi'] < 80)  # 强势未超买
].sort_values('volume_ratio').head(5)
```

**推荐数量**：2只（技术突破 + 放量）

---

#### 策略4：热点追踪（2只）

**筛选条件**：
```python
# 热门股票
hot_stocks = stocks[
    (stocks['sector_rank'] <= 10) &  # 板块排名前10
    (stocks['fund_inflow'] > 5000万) &  # 资金大幅流入
    (stocks['sentiment_score'] > 80) &  # 舆情热度高
    (stocks['change_pct'] > 3)  # 今日涨幅>3%
].sort_values('fund_inflow').head(5)
```

**推荐数量**：2只（资金流入 + 高热度）

---

#### 策略5：量化多因子（1只）

**因子体系**：
```python
# 综合评分
factor_score = (
    value_factor * 0.25 +  # 价值因子
    growth_factor * 0.30 +  # 成长因子（激进型权重高）
    tech_factor * 0.20 +  # 技术因子
    fund_factor * 0.15 +  # 资金因子
    sentiment_factor * 0.10  # 情绪因子
)

top_stock = stocks.sort_values('factor_score').head(1)
```

**推荐数量**：1只（综合评分最高）

### 第三步：风险控制（9:13-9:14）

**激进型风控规则**：

1. **仓位控制**：
   - 单只股票：10-15%（5-20万资金 → 0.5-3万/只）
   - 总仓位：最高80%（激进型可满仓）
   - 现金保留：20%（应急）

2. **止损设置**：
   - 单只股票止损：-8%（激进型稍宽）
   - 总仓位止损：-10%

3. **分散原则**：
   - 不同行业：至少3个行业
   - 不同市值：大中小盘均衡
   - 不同策略：价值/成长/技术混合

4. **市场环境判断**：
   - 大盘趋势：上涨/震荡/下跌
   - 板块轮动：热点切换
   - 风险提示：政策/外部因素

## 配置参数

```yaml
# config.yaml
task:
  id: daily_a_stock_recommendation
  schedule: "0 9:15 * * 1-5"  # 周一到周五 9:15
  
market:
  type: "A股"
  data_source: "akshare"  # akshare / sina / eastmoney
  
user:
  risk_preference: "aggressive"  # conservative / balanced / aggressive
  capital_range: "5-20万"
  
strategies:
  - name: "价值投资"
    enabled: true
    count: 2
  - name: "成长股"
    enabled: true
    count: 3
  - name: "技术分析"
    enabled: true
    count: 2
  - name: "热点追踪"
    enabled: true
    count: 2
  - name: "量化多因子"
    enabled: true
    count: 1
    
risk_control:
  single_position_max: 0.15  # 单只最大15%
  total_position_max: 0.80   # 总仓位最大80%
  stop_loss: -0.08           # 止损-8%
  
output:
  report_path: "知识库/股票推荐/"
  feishu_push: true
  feishu_user: "ou_d1b6454127592a07176fd36724723172"
```
