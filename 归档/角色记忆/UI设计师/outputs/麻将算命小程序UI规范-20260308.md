## 一、整体视觉风格

### 1.1 设计理念
**核心风格：** 新中式国风美学，融合传统麻将元素与现代简约设计

**设计关键词：**
- 典雅古朴
- 神秘玄妙
- 金玉满堂
- 传统文化与现代科技的平衡

### 1.2 视觉元素构成
- **底纹：** 淡雅云纹/祥云纹（透明度 5-10%）
- **边框：** 回纹/万字纹装饰线
- **点缀：** 金色祥云、铜钱纹、水墨晕染
- **质感：** 纸张质感 + 金属光泽点缀

## 五、界面布局方案

### 5.1 首页（Landing Page）

#### 组件层级
```
┌─────────────────────────────┐
│ [1] 顶部导航栏              │ ← 状态栏 + Logo + 分享
├─────────────────────────────┤
│ [2] 头部区域                │ ← 麻将牌动画 + 品牌标语
│     - 麻将牌旋转动画         │
│     - 标题"麻将算命"         │
│     - 副标题"天机玄妙"       │
├─────────────────────────────┤
│ [3] 功能入口卡片            │ ← 卡片式布局
│     ├─ 卡片1：生日算命       │
│     ├─ 卡片2：每日一签       │
│     └─ 卡片3：运势查询       │
├─────────────────────────────┤
│ [4] 今日运势快速预览        │ ← 横向滚动卡片
├─────────────────────────────┤
│ [5] 底部导航栏              │ ← 3个Tab（首页|历史|我的）
└─────────────────────────────┘
```

#### 交互说明
- **麻将牌动画：** 点击麻将牌进入生日算命
- **卡片点击：** 淡入淡出过渡动画（300ms）
- **今日运势：** 左右滑动查看更多

## 七、动画规范

### 7.1 动画时长
| 动画类型 | 时长 |
|---------|------|
| **页面切换** | 300ms |
| **按钮点击** | 150ms |
| **翻牌效果** | 600ms |
| **加载动画** | 1000ms |

### 7.2 缓动函数
```css
/* 标准缓动 */
ease: cubic-bezier(0.4, 0, 0.2, 1);

/* 弹性效果 */
bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);

/* 优雅过渡 */
smooth: cubic-bezier(0.25, 0.1, 0.25, 1);
```

### 7.3 核心动画
- **麻将翻牌：** 3D旋转 + 缩放
- **页面进入：** 向上滑入淡入
- **卡片展开：** 高度动画 + 内容淡入
- **按钮反馈：** 缩放 0.95
