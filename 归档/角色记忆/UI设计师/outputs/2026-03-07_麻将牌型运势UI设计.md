# 麻将牌型运势 UI 设计规范

**设计师：** UI 设计师  
**日期：** 2026-03-07  
**项目：** 麻将运势小程序

---

## 1. 页面位置

在现有「运势详情页」中新增一个模块

## 2. 新增模块：牌型运势卡片

### 布局结构

```
┌─────────────────────────────────────┐
│ 🎴 今日牌型运势                     │
├─────────────────────────────────────┤
│                                     │
│   🥇 条   87分  ⭐⭐⭐⭐⭐          │
│   🥈 筒   72分  ⭐⭐⭐⭐           │
│   🥉 万   58分  ⭐⭐⭐             │
│                                     │
├─────────────────────────────────────┤
│ 💡 运势解读：                       │
│ 今日木火当令，打条子牌运势最佳，    │
│ 建议打竹牌手气旺                  │
│                                     │
└─────────────────────────────────────┘
```

### 设计规范

**卡片尺寸：** 690rpx × 自适应  
**背景：** 白色 #FFFEF0  
**圆角：** 16rpx  
**边框：** 1rpx 流光金 #D4AF37  
**阴影：** 0 4rpx 16rpx rgba(0,0,0,0.08)

### 排名样式

| 排名 | 图标 | 背景色 | 文字色 |
|------|------|--------|--------|
| 🥇 第1 | 🎴 | #FFF8E7 (淡金) | #D4AF37 (金色) |
| 🥈 第2 | 🎴 | #F5F5F5 (灰白) | #666666 (灰色) |
| 🥉 第3 | 🎴 | #F5F5F5 (灰白) | #999999 (灰) |

### 分数显示

- 字号：36rpx，Bold
- 颜色：跟随排名色

### 星级评分

- 使用 ⭐ 字符
- 高分(80+): 5星 金色
- 中分(60-79): 3-4星 银色
- 低分(<60): 1-2星 灰色

### 运势解读

- 字号：26rpx
- 行高：1.6
- 颜色：#555555
- 背景：#FAFAFA 圆角 8rpx
- 内边距：16rpx

## 3. 动效设计

### 进厂动画
- 牌型卡片从下方滑入
- 时长：400ms
- 缓动：ease-out

### 分数跳动
- 数字从0滚动到目标分
- 时长：800ms
- 延迟：卡片滑入后200ms

## 4. 微信小程序适配

同其他组件，保持一致的 rpx 单位
