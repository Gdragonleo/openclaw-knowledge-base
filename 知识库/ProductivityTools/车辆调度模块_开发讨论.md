## 📝 开发注意事项

### 1. 数据校验规则
```python
# 车牌号格式校验
def validate_plate_number(plate: str) -> bool:
    """校验重庆市车牌格式"""
    import re
    pattern = r'^渝[A-H][A-Z0-9]{5,6}$'
    return bool(re.match(pattern, plate))

# 座位数范围校验
def validate_seat_count(count: int) -> bool:
    """校验座位数范围"""
    return 2 <= count <= 60

# 时间冲突检测
def detect_time_conflict(detail1: DispatchDetail, detail2: DispatchDetail) -> bool:
    """检测时间冲突"""
    return not (detail1.estimated_return_time < detail2.departure_time or
                detail2.estimated_return_time < detail1.departure_time)
```
