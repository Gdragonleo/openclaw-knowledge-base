## 💾 知识管理

### 存储规则
1. **分层存储**
   - 核心摘要 → MEMORY.md（<1KB/条）
   - 详细内容 → 知识库独立文件
   - 索引文件 → 团队索引.md

2. **文件命名**
   ```
   学习报告：YYYY-MM-DD_学习主题.md
   专题研究：研究主题_YYYY.md
   应用案例：案例名称.md
   ```

3. **定期归档**
   - 每周日自动归档
   - 保留最近1个月详细内容
   - 历史内容只保留摘要

### 防混乱机制
- ✅ **分层存储**：核心在MEMORY.md，详细在独立文件
- ✅ **按需检索**：使用memory_search快速定位
- ✅ **定期归档**：每周清理低价值内容
- ✅ **优先级管理**：保留高价值知识
