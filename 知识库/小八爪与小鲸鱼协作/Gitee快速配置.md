## 🚀 快速配置

### 1️⃣ 安装 Gitee

```bash
brew install gitee
```

**安装时间**：约2-3分钟

---

### 2️⃣ 配置用户信息

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

---

### 3️⃣ 克隆协作仓库

```bash
cd ~/
git clone https://gitee.com/whaleandcollab/agent-collaboration.git
```
