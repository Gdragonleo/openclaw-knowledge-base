## 一、算法框架设计

### 1.1 设计理念

麻将运势算法的设计理念是将中国传统算命智慧（四柱八字、六爻占卜、梅花易数）与现代游戏算法相结合，既传承文化，又增强游戏趣味性。

**核心原则**：
- **娱乐为主**：算法服务于游戏娱乐，不夸大预测功能
- **文化传承**：通过游戏传播传统算命文化
- **平衡公平**：运势增加趣味，但不破坏游戏公平性
- **透明可控**：向玩家解释原理，提供开关选项

### 1.2 整体架构

```
┌─────────────────────────────────────────────────┐
│                   输入层                         │
│  玩家信息 + 游戏时间 + 游戏状态                  │
└────────────────┬────────────────────────────────┘
                 ↓
┌─────────────────────────────────────────────────┐
│                   计算层                         │
│  八字计算 + 卦象计算 + 五行计算                  │
└────────────────┬────────────────────────────────┘
                 ↓
┌─────────────────────────────────────────────────┐
│                   映射层                         │
│  牌类映射 + 运势权重 + 牌义解读                  │
└────────────────┬────────────────────────────────┘
                 ↓
┌─────────────────────────────────────────────────┐
│                   输出层                         │
│  运势报告 + 牌局建议 + 可视化展示                │
└─────────────────────────────────────────────────┘
```

| 模块 | 功能 | 技术栈 |

```python
def calculate_bazi(birth_date, birth_time, gender, timezone):
    """
    八字计算核心流程
    
    输入：出生日期、时间、性别、时区
    输出：八字命盘、五行分布、日主强弱、喜用神
    """
    
    # Step 1: 转换为农历（可选，精确计算需要）
    lunar_date = solar_to_lunar(birth_date, timezone)
    
    # Step 2: 计算四柱
    year_pillar = calculate_year_pillar(birth_date.year, birth_date.month, birth_date.day)
    month_pillar = calculate_month_pillar(year_pillar, birth_date.month, birth_date.day)
    day_pillar = calculate_day_pillar(birth_date.year, birth_date.month, birth_date.day)
    hour_pillar = calculate_hour_pillar(day_pillar, birth_time.hour)
    
    # Step 3: 提取天干地支
    pillars = {
        'year': {'stem': year_pillar[0], 'branch': year_pillar[1]},
        'month': {'stem': month_pillar[0], 'branch': month_pillar[1]},
        'day': {'stem': day_pillar[0], 'branch': day_pillar[1]},
        'hour': {'stem': hour_pillar[0], 'branch': hour_pillar[1]}
    }
    
    # Step 4: 计算五行分布
    elements = calculate_elements_distribution(pillars)
    # 返回：{'木': 3, '火': 2, '土': 1, '金': 1, '水': 1}
    
    # Step 5: 判断日主强弱
    day_master = pillars['day']['stem']
    day_master_element = stem_to_element(day_master)
    month_branch = pillars['month']['branch']
    
    day_master_strength = analyze_day_master_strength(
        day_master_element, 
        month_branch, 
        elements
    )
    # 返回：'强' / '中' / '弱'
    
    # Step 6: 提取喜用神
    favorable_elements = determine_favorable_elements(day_master_element, day_master_strength)
    # 返回：['木', '金']（喜用神）
    
    unfavorable_elements = determine_unfavorable_elements(day_master_element, day_master_strength)
    # 返回：['水', '火']（忌神）
    
    return {
        'pillars': pillars,
        'elements': elements,
        'day_master': day_master,
        'day_master_element': day_master_element,
        'day_master_strength': day_master_strength,
        'favorable_elements': favorable_elements,
        'unfavorable_elements': unfavorable_elements
    }
```

### 3.2 卦象计算流程（梅花易数）

```python
def calculate_hexagram(game_time, optional_number=None):
    """
    卦象计算流程（梅花易数时间起卦法）
    
    输入：游戏时间、可选数字
    输出：本卦、变卦、动爻、卦象解读
    """
    
    # Step 1: 时间起卦
    if optional_number:
        # 数字起卦法
        upper_num = optional_number % 8
        lower_num = (optional_number + game_time.hour) % 8
        moving_line = (optional_number + game_time.hour) % 6
    else:
        # 时间起卦法
        year = game_time.year
        month = game_time.month
        day = game_time.day
        hour = game_time.hour
        
        upper_num = (year + month + day) % 8
        lower_num = (year + month + day + hour) % 8
        moving_line = (year + month + day + hour) % 6
    
    # Step 2: 数字转八卦
    trigram_map = {
        1: '乾', 2: '兑', 3: '离', 4: '震',
        5: '巽', 6: '坎', 7: '艮', 0: '坤'
    }
    
    upper_trigram = trigram_map[upper_num]
    lower_trigram = trigram_map[lower_num]
    
    # Step 3: 组合六十四卦
    hexagram = get_hexagram_by_trigrams(upper_trigram, lower_trigram)
    # 返回：{'name': '泰卦', 'symbol': '地天泰', 'meaning': '...'}
    
    # Step 4: 计算变卦
    changed_hexagram = calculate_changed_hexagram(hexagram, moving_line)
    
    # Step 5: 计算体用关系
    ti_yong_relation = analyze_ti_yong(hexagram, moving_line)
    # 返回：'用生体' / '体生用' / '用克体' / '体克用' / '比和'
    
    # Step 6: 卦象吉凶判断
    fortune = judge_hexagram_fortune(hexagram, changed_hexagram, ti_yong_relation)
    # 返回：'大吉' / '吉' / '中吉' / '小吉' / '平' / '小凶' / '凶' / '大凶'
    
    return {
        'original_hexagram': hexagram,
        'changed_hexagram': changed_hexagram,
        'moving_line': moving_line + 1,  # 转为1-6
        'ti_yong_relation': ti_yong_relation,
        'fortune': fortune
    }
```

### 3.3 运势综合计算流程

```python
def calculate_fortune_score(bazi_result, hexagram_result, game_info):
    """
    综合运势计算流程
    
    输入：八字结果、卦象结果、游戏信息
    输出：综合运势评分、五行运势、策略建议
    """
    
    # Step 1: 基础分（50分）
    base_score = 50
    
    # Step 2: 八字时辰运势调整（-20到+20）
    time_fortune = calculate_time_fortune(bazi_result, game_info['start_time'])
    
    # Step 3: 卦象运势调整（-15到+15）
    hexagram_fortune = calculate_hexagram_fortune_score(hexagram_result)
    
    # Step 4: 庄家加成（+5）
    dealer_bonus = 5 if game_info['is_dealer'] else 0
    
    # Step 5: 综合评分
    total_score = base_score + time_fortune + hexagram_fortune + dealer_bonus
    total_score = max(1, min(100, total_score))  # 限制在1-100
    
    # Step 6: 五行牌运计算
    element_fortunes = calculate_element_fortunes(bazi_result, hexagram_result)
    # 返回：{'木': 80, '火': 65, '土': 70, '金': 85, '水': 55}
    
    # Step 7: 策略建议生成
    strategy_advice = generate_strategy_advice(total_score, element_fortunes, hexagram_result)
    
    return {
        'overall_fortune': total_score,
        'element_fortunes': element_fortunes,
        'strategy_advice': strategy_advice
    }

def calculate_element_fortunes(bazi_result, hexagram_result):
    """
    计算五行牌运
    
    逻辑：
    1. 基础牌运 = 60（中性）
    2. 喜用神牌运 +20
    3. 忌神牌运 -20
    4. 卦象五行影响 +10/-10
    5. 随机波动 ±5
    """
    element_fortunes = {}
    
    base_fortune = 60
    
    for element in ['木', '火', '土', '金', '水']:
        fortune = base_fortune
        
        # 喜用神调整
        if element in bazi_result['favorable_elements']:
            fortune += 20
        elif element in bazi_result['unfavorable_elements']:
            fortune -= 20
        
        # 卦象五行影响
        hexagram_element_bonus = get_hexagram_element_bonus(hexagram_result, element)
        fortune += hexagram_element_bonus
        
        # 随机波动（保证趣味性）
        fortune += random.randint(-5, 5)
        
        # 限制范围
        fortune = max(1, min(100, fortune))
        
        element_fortunes[element] = fortune
    
    return element_fortunes
```

```python
# 主算法入口
def mahjong_fortune_algorithm(input_data):
    """
    麻将运势算法主入口
    
    输入：玩家信息、游戏信息
    输出：运势报告
    """
    
    # 1. 数据验证
    validate_input(input_data)
    
    # 2. 并行计算八字和卦象
    with ThreadPoolExecutor(max_workers=2) as executor:
        bazi_future = executor.submit(
            calculate_bazi, 
            input_data['player_info']['birth_date'],
            input_data['player_info']['birth_time'],
            input_data['player_info']['gender'],
            input_data['player_info']['timezone']
        )
        
        hexagram_future = executor.submit(
            calculate_hexagram,
            input_data['game_info']['start_time'],
            input_data['optional_info'].get('lucky_number')
        )
        
        bazi_result = bazi_future.result()
        hexagram_result = hexagram_future.result()
    
    # 3. 综合运势计算
    fortune_result = calculate_fortune_score(
        bazi_result, 
        hexagram_result, 
        input_data['game_info']
    )
    
    # 4. 生成完整报告
    report = generate_fortune_report(
        bazi_result,
        hexagram_result,
        fortune_result,
        input_data
    )
    
    return report
```

## 七、风险评估与应对

### 7.1 技术风险

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 算法复杂度高 | 性能瓶颈 | 缓存优化、查表法、并行计算 |
| 数据不准确 | 用户体验差 | 人工校验、持续优化、用户反馈 |
| 系统崩溃 | 服务中断 | 负载均衡、容灾备份、监控告警 |

### 7.2 业务风险

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 运势过于准确 | 破坏游戏公平 | 添加随机波动、限制运势影响 |
| 用户不信运势 | 功能闲置 | 文化教育、趣味解读、激励机制 |
| 迷信质疑 | 舆论风险 | 明确娱乐性质、文化传承定位 |

### 7.3 法律风险

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 涉嫌算命诈骗 | 法律纠纷 | 明确娱乐性质、不收费算命 |
| 未成年人沉迷 | 政策风险 | 实名认证、防沉迷系统 |
