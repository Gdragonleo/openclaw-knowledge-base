## 📚 知识库结构

### 1. 每日推荐记录/
记录每天的推荐股票和后续表现

- 格式：YYYY-MM-DD.md
- 内容：推荐股票、买入价、目标价、止损价、理由
- 跟踪：3日后表现、7日后表现、收益率

### 2. 市场环境/
记录市场特征和规律

- 市场阶段（牛市/熊市/震荡）
- 热点板块轮动
- 资金流向分析
- 历史规律总结

### 3. 策略评估/
评估各策略的效果

- 价值投资策略效果
- 成长股策略效果
- 技术分析策略效果
- 热点追踪策略效果
- 量化多因子策略效果

### 4. 特殊事件/
记录影响市场的重要事件

- 政策影响
- 外部冲击
- 突发事件
- 应对策略

### 5. 案例库/
经典案例分析

- 成功案例
- 失败案例
- 教训总结
- 经验提炼

### 6. 规则库/
投资纪律和规则

- 操作纪律
- 风控规则
- 市场规律
- 策略选择规则
