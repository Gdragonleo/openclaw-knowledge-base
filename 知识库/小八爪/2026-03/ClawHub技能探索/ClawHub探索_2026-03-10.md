## ✅ 已学习并安装的技能（2个）

### 1. 🕷️ Crawl4AI Skill（crawl4ai-skill）

**功能**：集搜索、爬取、省Token、热门站登录为一身的智能工具

**核心能力**：
- 🔍 DuckDuckGo搜索（免API key）
- 🕷️ 智能全站爬取（自动识别sitemap/llms-full.txt）
- 📝 LLM优化输出（Fit Markdown，去冗余）
- 🔐 登录态爬取（Twitter/X、小红书）
- 🐦 推文提取（支持引用推文）

**安装路径**：`/Users/danxiong/.openclaw/workspace/skills/crawl4ai-skill`

**依赖**：
- Python >= 3.9
- Playwright + Chromium

**使用场景**：
- 爬取需要登录的平台（Twitter/X、小红书）
- 全站文档爬取并转为Markdown
- 搜索并爬取相关网页
- 节省Token的LLM优化输出

**安全注意事项**：
- Session cookies存储在本地 `~/.crawl4ai-skill/sessions/<platform>.json`
- 建议使用一次性/测试账号
- 完成后清除登录信息：`crawl4ai-skill session-clear`

**主要命令**：
```bash
# 搜索
crawl4ai-skill search "python web scraping"

# 爬取单页
crawl4ai-skill crawl https://example.com

# 爬取全站
crawl4ai-skill crawl-site https://docs.example.com --max-pages 50

# 登录Twitter
crawl4ai-skill login twitter --cookies "auth_token=xxx; ct0=yyy"

# 登录态爬取
crawl4ai-skill crawl-with-login https://x.com/elonmusk -p twitter --extract-tweets
```

**优点**：
- 一站式解决方案（搜索+爬取+优化）
- 节省Token成本
- 支持热门平台登录态爬取
- 无需API key

**缺点**：
- 需要Python环境和Playwright
- 登录态需要手动获取Cookie
- 安全性需要注意（Cookie存储）

**刘氏集团适用性**：⭐⭐⭐⭐⭐
- 可以爬取竞品信息、行业报告
- 可以爬取小红书等平台的市场情报
- 节省Token成本，提高效率

### 2. 🔍 Multi Search Engine（multi-search-engine）

**功能**：多搜索引擎集成，支持17个搜索引擎（8个中文+9个国际）

**核心能力**：
- 支持17个搜索引擎（无需API key）
- 高级搜索操作符（site:、filetype:等）
- 时间过滤器
- 隐私搜索引擎
- WolframAlpha知识查询

**安装路径**：`/Users/danxiong/.openclaw/workspace/skills/multi-search-engine`

**支持的搜索引擎**：

**中文（8个）**：
1. 百度
2. Bing中文
3. Bing国际
4. 360搜索
5. 搜狗
6. 微信搜狗
7. 今日头条
8. 集思录

**国际（9个）**：
1. Google
2. Google HK
3. DuckDuckGo
4. Yahoo
5. Startpage
6. Brave
7. Ecosia
8. Qwant
9. WolframAlpha

**使用场景**：
- 多引擎搜索对比
- 特定站点搜索
- 文件类型搜索（PDF、DOC等）
- 隐私搜索
- 数学计算、货币转换（WolframAlpha）

**主要用法**：
```bash
# 基础搜索
web_fetch({"url": "https://www.google.com/search?q=python+tutorial"})

# 站点搜索
web_fetch({"url": "https://www.google.com/search?q=site:github.com+react"})

# 文件类型
web_fetch({"url": "https://www.google.com/search?q=machine+learning+filetype:pdf"})

# 时间过滤（过去一周）
web_fetch({"url": "https://www.google.com/search?q=ai+news&tbs=qdr:w"})

# 隐私搜索
web_fetch({"url": "https://duckduckgo.com/html/?q=privacy+tools"})

# DuckDuckGo Bangs快捷方式
web_fetch({"url": "https://duckduckgo.com/html/?q=!gh+tensorflow"})

# 知识计算
web_fetch({"url": "https://www.wolframalpha.com/input?i=100+USD+to+CNY"})
```

**高级操作符**：
- `site:` - 站内搜索
- `filetype:` - 文件类型
- `""` - 精确匹配
- `-` - 排除词
- `OR` - 或运算

**时间过滤器**：
- `tbs=qdr:h` - 过去1小时
- `tbs=qdr:d` - 过去1天
- `tbs=qdr:w` - 过去1周
- `tbs=qdr:m` - 过去1个月
- `tbs=qdr:y` - 过去1年

**优点**：
- 无需API key
- 支持17个搜索引擎
- 包含中文和国际引擎
- 支持高级搜索语法
- 隐私保护选项

**缺点**：
- 需要网络访问（某些引擎可能被墙）
- 依赖web_fetch工具
- 无并发搜索能力

**刘氏集团适用性**：⭐⭐⭐⭐⭐
- 可以多引擎搜索对比结果
- 支持中文和国际搜索引擎
- 适合深度调研和市场分析
- 隐私搜索引擎保护数据安全
