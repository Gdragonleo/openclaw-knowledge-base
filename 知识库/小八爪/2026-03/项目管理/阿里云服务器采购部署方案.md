### 购买步骤

#### 1. 登录阿里云（2分钟）
- 访问：https://www.aliyun.com
- 登录/注册账号
- 完成实名认证（如果需要）

#### 2. 进入ECS控制台（1分钟）
- 控制台首页 → 云服务器ECS
- 点击"创建实例"

#### 3. 选择配置（5分钟）
**基础配置**：
- 付费模式：按量付费
- 地域：华北2（北京）cn-beijing
- 可用区：随机分配（通用型）

**实例配置**：
- 实例规格：2 vCPU / 4 GiB
- CPU架构：x86计算型
- 实例类型：计算型 c6
- 实例规格：ecs.c6.large

**存储配置**：
- 系统盘：40GB ESSD
- 数据盘：不需要（节省成本）

**网络配置**：
- 网络类型：专有网络
- 分配公网IP：勾选
- 带宽计费：按流量计费
- 带宽峰值：5Mbps

**系统配置**：
- 镜像：Ubuntu 22.04 64位
- 登录凭证：自定义密码
  - 用户名：root
  - 密码：（设置一个强密码，例如：Whale@2026!）

**安全组配置**：
- 默认开放：22（SSH）、80（HTTP）、443（HTTPS）、3000（OpenClaw）
  - 或者：选择"自动创建并允许所有流量"（测试阶段）

#### 4. 确认订单（2分钟）
- 检查配置信息
- 确认价格（预估¥60-100/月）
- 勾选"我已阅读并同意《云服务器ECS服务条款》"
- 点击"创建实例"

#### 5. 等待实例创建（3-5分钟）
- 实例创建中...
- 创建完成后会显示：
  - 公网IP地址
  - 内网IP地址
  - 实例ID

#### 6. 连接测试（1分钟）
```bash
# 在本地终端测试连接
ssh root@你的公网IP

# 输入刚才设置的密码
# 成功登录后显示：
Welcome to Ubuntu 22.04 LTS
```

## 🚀 第二步：服务器环境配置（20-30分钟）

### 连接服务器后执行以下命令

#### 1. 更新系统（5分钟）
```bash
# 更新软件包列表
apt update

# 升级所有软件包
apt upgrade -y

# 安装基础工具
apt install -y curl wget git vim htop net-tools
```

#### 2. 安装Node.js 18（5分钟）
```bash
# 添加NodeSource仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -

# 安装Node.js
apt install -y nodejs

# 验证安装
node -v   # 应该显示 v18.x.x
npm -v    # 应该显示 9.x.x
```

#### 3. 安装OpenClaw（10分钟）
```bash
# 全局安装OpenClaw
npm install -g openclaw

# 验证安装
openclaw --version

# 初始化配置
openclaw config init
```

#### 4. 配置防火墙（5分钟）
```bash
# 开放必要端口
ufw allow 22/tcp    # SSH
ufw allow 80/tcp    # HTTP
ufw allow 443/tcp   # HTTPS
ufw allow 3000/tcp  # OpenClaw

# 启用防火墙
ufw enable

# 查看状态
ufw status
```

#### 5. 配置SSH密钥登录（5分钟，可选）
```bash
# 在本地Mac生成SSH密钥（如果还没有）
# 本地执行：
ssh-keygen -t rsa -b 4096 -C "xiaobazhua@aliyun"

# 复制公钥到服务器
# 本地执行：
ssh-copy-id root@你的公网IP

# 测试密钥登录
ssh root@你的公网IP

# 禁用密码登录（可选，更安全）
# 服务器上执行：
vim /etc/ssh/sshd_config
# 找到：PasswordAuthentication yes
# 改为：PasswordAuthentication no
# 重启SSH服务
systemctl restart sshd
```

## 🐳 第三步：部署小鲸鱼（30-40分钟）

### 1. 创建工作目录
```bash
# 创建小鲸鱼工作目录
mkdir -p ~/.openclaw/workspace

# 进入目录
cd ~/.openclaw/workspace
```

### 2. 克隆知识库（10分钟）
```bash
# 克隆你的知识库（如果有）
git clone https://github.com/Gdragonleo/openclaw-knowledge-base.git .

# 或者创建新的workspace
mkdir -p 知识库/小鲸鱼
```

### 3. 配置小鲸鱼（10分钟）
```bash
# 创建配置文件
openclaw config set agent.name "小鲸鱼"
openclaw config set agent.model "kimi-k2.5"
openclaw config set agent.role "首席技术官"

# 配置飞书（重要！）
openclaw config set feishu.appId "你的飞书AppID"
openclaw config set feishu.appSecret "你的飞书AppSecret"

# 配置协作中心
openclaw config set collaboration.center "https://bcnuqjt2v034.feishu.cn/base/CnBNbHdZnaePv4s4StLcONDXngc"
openclaw config set collaboration.tableId "tblcxFySU7AaXjee"
```

### 4. 测试小鲸鱼（5分钟）
```bash
# 启动小鲸鱼
openclaw start

# 查看状态
openclaw status

# 测试协作中心连接
openclaw test collaboration

# 测试飞书连接
openclaw test feishu
```

### 5. 配置自动启动（5分钟）
```bash
# 安装PM2
npm install -g pm2

# 创建启动脚本
cat > ~/.openclaw/start-whale.sh << 'EOF'
#!/bin/bash
cd ~/.openclaw/workspace
openclaw start
EOF

chmod +x ~/.openclaw/start-whale.sh

# 使用PM2管理
pm2 start ~/.openclaw/start-whale.sh --name "whale"

# 保存PM2配置
pm2 save

# 设置开机自启
pm2 startup
```

## 🆘 遇到问题怎么办

### 问题1：无法SSH连接
**解决方案**：
```bash
# 检查安全组规则
# 在阿里云控制台 → 云服务器ECS → 实例 → 安全组
# 确认22端口已开放
```

### 问题2：Node.js安装失败
**解决方案**：
```bash
# 手动安装
wget https://nodejs.org/dist/v18.17.0/node-v18.17.0-linux-x64.tar.xz
tar -xzf node-v18.17.0-linux-x64.tar.xz
cd node-v18.17.0-linux-x64
./configure
make
make install
```

### 问题3：飞书连接失败
**解决方案**：
```bash
# 检查配置
openclaw config get feishu

# 测试API连接
curl -X POST "https://open.feishu.cn/open-apis/auth/v3/app_access_token" \
  -H "Authorization: Basic base64(AppID:AppSecret)"
```

### 问题4：PM2启动失败
**解决方案**：
```bash
# 查看错误日志
pm2 logs whale --err

# 重启PM2
pm2 restart whale

# 或直接启动
cd ~/.openclaw/workspace
openclaw start
```
