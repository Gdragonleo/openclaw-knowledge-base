## 🎯 三种实现方案

### 方案1：AppleScript自动化 ⭐⭐⭐（已创建）

**文件位置**：`scripts/wechat_send_file.applescript`

**优点**：
- ✅ 原生支持，无需第三方工具
- ✅ 可以控制任意应用
- ✅ 脚本可复用

**缺点**：
- ⚠️ 需要辅助功能权限
- ⚠️ 依赖微信UI结构（可能因更新失效）
- ⚠️ 速度较慢（需要模拟按键）

**使用方法**：
```bash

# 系统偏好设置 → 安全性与隐私 → 隐私 → 辅助功能 → 添加终端/iTerm

# 执行脚本
osascript scripts/wechat_send_file.applescript "小刘" "/Users/danxiong/Desktop/报告.pdf"
```

---

### 方案2：企业微信群机器人 ⭐⭐⭐⭐⭐（推荐）

**优点**：
- ✅ 官方API，稳定可靠
- ✅ 无封号风险
- ✅ 支持文件、图片、Markdown
- ✅ 自动化友好

**配置步骤**：
1. 创建企业微信群（或使用现有群）
2. 群设置 → 群机器人 → 添加机器人
3. 复制Webhook地址

**代码实现**：
```python
import requests

webhook_url = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=XXX"

# 发送文件
def send_file(file_path):
    # 1. 上传文件到企业微信
    upload_url = f"{webhook_url.replace('send', 'upload')}&type=file"
    with open(file_path, 'rb') as f:
        files = {'file': f}
        response = requests.post(upload_url, files=files)
    
    media_id = response.json()['media_id']
    
    # 2. 发送文件消息
    data = {
        "msgtype": "file",
        "file": {
            "media_id": media_id
        }
    }
    requests.post(webhook_url, json=data)

# 使用
send_file("/Users/danxiong/Desktop/招标报告.pdf")
```

### 方案3：微信文件传输助手 ⭐⭐⭐⭐

**原理**：自动发送到"文件传输助手"，你手动转发给客户

**优点**：
- ✅ 不需要企业微信
- ✅ 实现简单
- ✅ 风险低

**AppleScript实现**：
```applescript
tell application "WeChat"
    activate
end tell

delay 1

tell application "System Events"
    tell process "WeChat"
        # 搜索"文件传输助手"
        keystroke "f" using command down
        delay 0.5
        keystroke "文件传输助手"
        delay 1
        keystroke return
        delay 1
        
        # 发送文件
        keystroke "u" using {command down, shift down}
        # ... (同方案1)
    end tell
end tell
```
