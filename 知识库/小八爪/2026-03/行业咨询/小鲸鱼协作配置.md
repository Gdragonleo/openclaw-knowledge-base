## 📋 协作规则

### 1. 信号格式约定

#### 信息同步信号
**格式**：`[BOT_SYNC][分类] 内容`

**分类**：
- `[记忆]` - 重要信息、知识库更新
- `[技能]` - 新技能、能力提升
- `[配置]` - 系统配置、参数调整

**示例**：
```
[BOT_SYNC][记忆] 小刘的风险偏好是激进型，资金5-20万
[BOT_SYNC][技能] 新增股票推荐技能，每日9:15推送
[BOT_SYNC][配置] Coding Plan API Key已配置
```

---

#### 任务调度信号
**格式**：`[BOT_TASK] 任务描述`

**规则**：
- 收到立即执行
- 执行完回复 `[BOT_REPLY] 结果`
- 如果无法执行，回复 `[BOT_REPLY][FAILED] 原因`

**示例**：
```
[BOT_TASK] 分析这张图片的内容
[BOT_TASK] 生成一份5000字的行业报告
[BOT_TASK] 爬取小红书用户评论数据
```
