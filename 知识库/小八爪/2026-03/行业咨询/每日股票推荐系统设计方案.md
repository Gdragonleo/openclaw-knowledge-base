## 🎯 系统目标

**核心功能**：
1. 每日自动分析市场数据
2. 基于策略筛选推荐股票
3. 生成详细推荐报告

## 🎯 推荐策略设计

### 策略1：价值投资策略（保守型）

**筛选标准**：
```
1. PE（市盈率） < 15
2. PB（市净率） < 2
3. ROE（净资产收益率） > 15%
4. 股息率 > 3%
5. 连续3年盈利增长
6. 市值 > 100亿（流动性好）
```

**适合**：
- 风险偏好：保守
- 投资周期：中长线（6-12个月）
- 预期收益：10-20%/年

**典型股票**：
- 银行股（招商银行、平安银行）
- 公用事业（长江电力）
- 消费蓝筹（贵州茅台、五粮液）

---

### 策略2：成长股策略（激进型）

**筛选标准**：
```
1. 营收增长率 > 30%
2. 净利润增长率 > 30%
3. 行业前景好（新能源/半导体/医药）
4. 市值 < 500亿（成长空间大）
5. 研发投入占比 > 5%
6. 机构持仓比例 > 20%
```

**适合**：
- 风险偏好：激进
- 投资周期：中短线（3-6个月）
- 预期收益：30-50%/年

**典型股票**：
- 新能源（宁德时代、比亚迪）
- 半导体（北方华创、中微公司）
- 医药（药明康德、恒瑞医药）

---

### 策略3：技术分析策略（平衡型）

**筛选标准**：
```
1. 突破关键阻力位
2. 5日均线上穿20日均线（金叉）
3. MACD金叉
4. RSI在50-70之间（强势但未超买）
5. 成交量放大 > 1.5倍均量
6. 资金净流入 > 5000万
```

**适合**：
- 风险偏好：平衡
- 投资周期：短线（1-4周）
- 预期收益：10-30%/次

**典型操作**：
- 突破买入
- 回调加仓
- 止损位设置

---

### 策略4：热点追踪策略（激进型）

**筛选标准**：
```
1. 涨停板或接近涨停
2. 板块整体上涨 > 3%
3. 龙虎榜净买入 > 1亿
4. 北向资金流入 > 5亿
5. 市场热度排名前10%
6. 新闻正面舆情
```

**适合**：
- 风险偏好：激进
- 投资周期：超短线（1-5天）
- 预期收益：5-20%/次

**风险提示**：
- 高风险高收益
- 需要快进快出
- 严格止损

---

### 策略5：量化多因子策略（专业型）

**因子体系**：

**价值因子（30%）**：
- PE、PB、PS、PCF
- 股息率、PEG

**成长因子（25%）**：
- 营收增长率
- 净利润增长率
- ROE变化

**技术因子（20%）**：
- 动量因子（过去1个月涨幅）
- 反转因子（过去6个月跌幅）
- 波动率

**资金因子（15%）**：
- 北向资金流入
- 龙虎榜净买入
- 机构持仓变化

**情绪因子（10%）**：
- 舆情热度
- 搜索指数
- 社交媒体讨论度

**综合评分**：
```
总分 = Σ(因子得分 × 因子权重)
推荐 = 总分排名前10的股票
```

## 🔧 技术实现方案

### 方案1：基于现有技能（快速实现）⭐

**技术栈**：
- **数据爬取**：tender-browser-scraper（已有的招标爬虫技能）
- **数据处理**：Python + pandas
- **报告生成**：Markdown模板
- **推送通知**：飞书消息

**优点**：
- ✅ 快速实现（1-2天）
- ✅ 成本低（免费）
- ✅ 易于维护

**缺点**：
- ⚠️ 数据可能不稳定
- ⚠️ 功能相对简单

---

### 方案2：基于免费API（推荐）

**技术栈**：
- **数据源**：AkShare（Python库，免费）
- **数据处理**：Python + pandas + talib
- **策略引擎**：自定义策略函数
- **报告生成**：Jinja2模板
- **推送通知**：飞书API

**优点**：
- ✅ 数据稳定（官方库）
- ✅ 功能完善
- ✅ 可扩展性好

**实现步骤**：
```python
# 1. 安装依赖
pip install akshare pandas talib jinja2

# 2. 数据获取
import akshare as ak
stock_data = ak.stock_zh_a_spot_em()  # A股实时行情

# 3. 策略筛选
def filter_value_stocks(data):
    return data[
        (data['pe'] < 15) & 
        (data['pb'] < 2) & 
        (data['roe'] > 15)
    ]

# 4. 生成报告
def generate_report(stocks):
    template = """
    # 📊 每日股票推荐
    日期：{{ date }}
    
    ## 推荐股票
    {% for stock in stocks %}
    {{ loop.index }}. {{ stock.name }} ({{ stock.code }})
       - PE: {{ stock.pe }}
       - PB: {{ stock.pb }}
       - 推荐理由: {{ stock.reason }}
    {% endfor %}
    """
    return render(template, stocks=stocks)
```
