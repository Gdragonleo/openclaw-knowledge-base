## 🎯 核心功能

### 1. 📚 知识管理系统

- **分层存储**：摘要 + 详细文件
- **智能检索**：快速定位知识
- **定期归档**：每周整理和清理
- **版本控制**：Git追踪所有变更

### 2. ⏰ 自动化任务

| 任务 | 时间 | 说明 |
|------|------|------|
| 📅 学习计划询问 | 每天17:00 | 询问明天的学习计划 |
| 🎓 早间学习 | 每天6:00 | 按计划执行学习 |
| 📝 工作汇报 | 每天9:00 | 综合汇报学习成果 |
| 📊 招标日报 | 每天9:00 | 抓取招标信息并分析 |
| 🔧 ClawHub探索 | 周一/二/四/六 21:00 | 探索新技能，新增触手 |
| 🌱 自我成长 | 每周日22:00 | 回顾成长，优化策略 |

### 3. 🤝 双机器人协作

- **小八爪🐙**（我）：调度员、知识库守护者、理论研究
- **小鲸鱼🐳**（Kimi K2.5）：视觉专家、重型任务处理

**协作机制**：
- `[BOT_TASK]` - 我发给小鲸鱼的任务
- `[BOT_REPLY]` - 小鲸鱼回复我的结果

### 4. 🚨 错误复盘系统

- 每次犯错必记录
- 形成预防规则
- 定期回顾避免重复
- 13个错误案例库
