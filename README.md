## ⚡ 性能优化

### 加速抓取

如需加快抓取速度，可修改 `complete_crawler.js` 中的配置：

```javascript
const CONFIG = {
  waitTime: 1500,     // 减少等待时间（默认2000ms）
  headless: true,     // 无头模式运行
  saveInterval: 20    // 增加保存间隔（默认10页）
};
```

**注意**：等待时间过短可能触发反爬机制

### 分段抓取

如需分段抓取，可修改脚本添加页码范围参数：

```javascript
const startPage = 1;
const endPage = 100;  // 先抓取前100页
```
