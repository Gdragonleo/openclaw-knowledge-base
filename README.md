## 🎓 知识管理理念

### **采用Obsidian方法**：
```
✅ 双向链接 [[文件名]]
✅ MOC（Map of Content）导航
✅ 原子化笔记（一个文件一个主题）
✅ 标签系统 #标签
✅ 知识网络化（非层级化）
```

### **为什么用Obsidian？**
```
✅ 知识关联：自动发现隐含联系
✅ 快速导航：MOC + 链接 = 秒级到达
✅ 思考深度：网络化思考
✅ 知识复用：链接复用而非复制
```
